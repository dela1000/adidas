# Adidas Code Challenge
> A React.js and Node.js Example

## Development

Open two Terminals:
On Terminal 1:

```bash
npm i && npm start
```
to install npm dependencies and run webpack.

On Terminal 2:

```bash
node server.js
```

________

To initiate node.js API server.

Find the page at 
```bash
http://localhost:8080/
```# adidas
